# Gleaner 使用文档

Gleaner 是一个基于 GitHub 的只读知识库阅读器，灵感来自 [Obsidian](https://obsidian.md)。它让你可以直接在浏览器中阅读 GitHub 仓库里的 Markdown 文件，支持 `[[双向链接]]`、知识图谱和多仓库管理。

> 无需后端，纯前端 SPA，数据缓存在浏览器 IndexedDB 中。
