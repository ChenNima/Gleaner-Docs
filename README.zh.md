# Gleaner 使用文档

Gleaner 在浏览器中直接阅读 GitHub 仓库的 Markdown 文件，支持 `[[双向链接]]`、知识图谱和多仓库聚合。

> 纯前端，无后端依赖，数据缓存在浏览器 IndexedDB 中。
