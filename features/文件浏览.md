# 文件浏览

Gleaner 的核心体验是一个三栏布局：左侧文件树、中间内容区、右侧链接面板。

## 文件树

左侧边栏展示所有已同步仓库的文件目录结构。

- **文件夹** 可以展开/折叠
- **文件** 按字母排序，文件夹排在文件前面
- 点击文件即可在中间区域渲染 Markdown
- 当前打开的文件会高亮显示

### 多仓库

如果你配置了多个仓库（参见 [[多仓库管理]]），文件树会按仓库分组显示，每个仓库作为一个独立的可折叠区域。每个仓库旁边会显示同步状态和文件计数。

### 折叠全部

点击文件树顶部的折叠按钮可以一次性收起所有展开的目录。

## 内容区

中间区域渲染 Markdown 内容，支持：

- 标准 Markdown 语法（标题、列表、表格、代码块等）
- GFM（GitHub Flavored Markdown）扩展
- 代码高亮（自动语言检测）
- `[[wikilinks]]` — 详见 [[双向链接]]
- 内嵌 HTML（包括 `<video>` 标签）
- 数学公式（如果源文件包含）

### 面包屑导航

内容区顶部显示当前文件的路径，你可以点击路径中的任意部分跳转到文件树中对应的目录。

### 文档主题

你可以通过面包屑右侧的主题选择器切换渲染风格，详见 [[文档主题]]。

## 右侧面板

右侧面板提供两个选项卡：

- **链接** — 显示当前文件的 [[双向链接|反向链接]] 和出链
- **图谱** — 显示当前文件的局部 [[知识图谱|关系图]]

## 侧边栏控制

- 点击顶部导航栏的面板按钮可以显示/隐藏左右侧边栏
- 桌面端可以拖拽侧边栏边缘调整宽度
- 移动端侧边栏以浮层形式弹出，不占用内容空间

## 快捷操作

- 使用 [[全局搜索|Cmd+K]] 快速定位文件
- 使用浏览器前进/后退按钮在文件之间导航
- 更多操作见 [[快捷键]]
