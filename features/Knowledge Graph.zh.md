# 知识图谱

知识图谱以力导向图的形式展示所有文件及其连接关系。

## 全局图谱

通过导航栏访问全局图谱。它显示：

- **节点** — 每个 Markdown 文件显示为一个节点
- **边** — 文件之间的每条链接（wikilink 或标准 Markdown 链接）显示为一条边
- **颜色编码** — 节点按所属仓库着色，便于区分不同仓库的文件

### 交互操作

- **悬停** — 将鼠标悬停在节点上会高亮该节点及其直接连接，其他节点变暗
- **点击** — 点击节点可导航到该文件的内容视图
- **滚轮缩放** — 使用鼠标滚轮进行缩放
- **拖动** — 拖动单个节点可重新定位，拖动背景可平移视图

### 过滤器

图谱工具栏提供过滤选项：

- **外部链接开关** — 显示或隐藏指向外部 URL 的边
- **仓库可见性** — 逐个切换仓库的显示/隐藏，聚焦于知识库的特定子集

## 局部图谱

除了全局图谱页面外，在查看任何文件时，右侧面板中还提供**局部图谱**。它显示：

- 当前文件作为中心节点
- 所有 1 度连接范围内的文件（链接到当前文件或被当前文件链接的文件）
- 这些文件之间的边

局部图谱可以在不离开当前页面的情况下查看文件的直接上下文。

## 使用场景

| 目标 | 图谱如何帮助 |
|------|------------|
| 查找孤立笔记 | 寻找没有边的节点——这些文件没有任何链接关系 |
| 识别枢纽文档 | 大型、高度连接的节点是关键参考文档 |
| 追踪路径 | 沿着节点之间的边查看两个主题的关联方式 |
| 仓库概览 | 颜色编码显示内容在各仓库间的分布情况 |
