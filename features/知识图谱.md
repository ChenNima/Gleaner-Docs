# 知识图谱

知识图谱以力导向图（Force-directed Graph）的形式可视化你的笔记网络。每个节点代表一篇文档，每条连线代表一个 [[双向链接]]。

## 全局图谱

点击顶部导航栏的网络图标，进入全局图谱视图。你会看到：

- **所有已同步的 Markdown 文件** 作为节点
- **wikilinks 和 Markdown 链接** 作为连线
- **不同仓库的文件** 使用不同颜色区分（参见 [[多仓库管理]]）

### 交互操作

| 操作 | 效果 |
|------|------|
| 鼠标悬停 | 高亮节点及其直接连接 |
| 点击节点 | 跳转到对应文件 |
| 滚轮 | 缩放 |
| 拖拽空白区域 | 平移画布 |
| 拖拽节点 | 移动节点位置 |

### 过滤选项

图谱页面左上角显示统计信息和过滤控件：

- **笔记数** / **链接数** — 当前图谱的节点和边数量
- **外部链接** 开关 — 是否显示指向外部网站的链接节点
- **仓库过滤** — 点击仓库名称可以隐藏/显示特定仓库的节点

## 局部图谱

在阅读单篇文件时，右侧面板的「图谱」选项卡显示以当前文件为中心的局部关系图。

局部图谱只展示当前文件的直接连接（一度关系），更聚焦地展示当前笔记的上下文。

## 图谱的用途

知识图谱不仅好看，还很实用：

- **发现孤立笔记** — 没有连线的节点可能需要更多链接
- **识别核心概念** — 连线最多的节点是知识网络的枢纽
- **探索关联** — 通过图谱发现两个看似无关的笔记之间的路径
- **仓库概览** — 快速了解一个仓库的内容结构和主题分布

## 与其他功能的关系

- 图谱中的连线来自 [[双向链接]] 的解析结果
- 点击图谱节点会进入 [[文件浏览]] 模式
- [[多仓库管理|多仓库]] 场景下，不同仓库用不同颜色区分
- 使用 [[全局搜索]] 可以更快定位特定文件
