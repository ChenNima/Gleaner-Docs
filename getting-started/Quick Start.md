# Quick Start

Get started with Gleaner in 3 simple steps.

## Step 1: Open Gleaner

Visit the Gleaner deployment URL. On first visit, you'll be redirected to the [First Setup](./First%20Setup.md) page.

## Step 2: Add a Repository

In the Settings page under the "Repositories" tab:

1. Enter a GitHub repository in `owner/repo` format (e.g. `ChenNima/Gleaner-Docs`)
2. Optionally add a label for easy identification
3. Click "Add"
4. Click "Save & Sync"

> For detailed repository options, see [Repo Config](../advanced/Repo%20Config.md).

## Step 3: Start Reading

Once sync completes, the file tree appears in the left sidebar. Click any `.md` file to start reading.
