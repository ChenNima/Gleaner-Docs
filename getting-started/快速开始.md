# 快速开始

只需 3 步，你就可以在浏览器中阅读 GitHub 上的 Markdown 知识库。

## 第 1 步：打开 Gleaner

访问 Gleaner 的部署地址。首次打开会自动跳转到 [[首次配置]] 页面。

## 第 2 步：添加仓库

在设置页面的「仓库」选项卡中：

1. 输入 GitHub 仓库地址，格式为 `owner/repo`（例如 `ChenNima/Gleaner-Docs`）
2. 可选：填写一个易于辨认的标签
3. 点击「添加」
4. 点击「保存并同步」

> 详细的仓库配置选项请参见 [[仓库高级配置]]。

## 第 3 步：开始阅读

同步完成后，左侧边栏会出现文件树。点击任意 `.md` 文件即可开始阅读。
