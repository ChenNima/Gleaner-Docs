# 多语言 (i18n)

Gleaner 的用户界面支持多种语言。你可以选择跟随系统语言，或者明确设置偏好语言。

## 语言选项

| 选项 | 行为 |
|------|------|
| **跟随系统** | Gleaner 自动检测浏览器/操作系统的语言设置并使用 |
| **English** | 无论系统设置如何，强制界面使用英文 |
| **中文** | 无论系统设置如何，强制界面使用中文 |

## 系统检测逻辑

当设置为「跟随系统」时，Gleaner 读取浏览器的 `navigator.language` 属性：

- 如果值以 `zh` 开头（如 `zh-CN`、`zh-TW`、`zh`），则使用中文
- 其他所有值默认使用英文

## 影响范围

语言设置应用于**所有 UI 元素**，包括：

- 导航栏标签
- 侧边栏标题和控件
- 设置页面的标签和描述
- 搜索对话框的占位文本
- 状态消息和同步进度
- 按钮标签和工具提示

**注意：**语言设置**不会**影响文档内容。你的 Markdown 文件会按原样渲染，无论它们以何种语言编写。

## 持久化

语言偏好存储在 `localStorage` 中，跨浏览器会话持久化。清除浏览器数据或 localStorage 会将其重置为默认的「跟随系统」行为。
