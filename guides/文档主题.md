# 文档主题

Gleaner 提供多种 Markdown 渲染主题，你可以根据阅读偏好选择不同的视觉风格。

## 可用主题

| 主题 | 风格 | 适合场景 |
|------|------|----------|
| **GitHub** | 简洁熟悉 | 习惯 GitHub 风格的用户 |
| **Obsidian** | 紧凑的维基风格 | 密集信息浏览 |
| **Academic** | 衬线字体，纸质质感 | 长文阅读、学术文档 |
| **Notion** | 宽敞现代 | 舒适的视觉体验 |
| **Newsprint** | 经典阅读体验 | 文字密集的文档 |

## 切换主题

在 [[文件浏览|文件阅读页面]] 的面包屑导航右侧，点击主题选择器图标即可切换。

选择会自动保存，下次打开时保持你的选择。

## 明暗模式

除了文档主题，Gleaner 还支持系统级的明暗模式切换：

- 点击导航栏的太阳/月亮图标切换
- 代码高亮会自动适配当前模式
- 所有文档主题都同时支持明暗两种模式

## 代码高亮

代码块使用 `highlight.js` 自动检测语言并高亮，支持数十种编程语言。每个代码块右上角有复制按钮。

明暗模式下会自动切换不同的代码高亮配色方案。
