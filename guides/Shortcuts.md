# Keyboard Shortcuts

Gleaner supports keyboard shortcuts for efficient navigation. This page lists all available shortcuts and navigation tips.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| **Cmd+K** (Mac) / **Ctrl+K** (Windows/Linux) | Open the [Search](../features/Search.md) dialog |

## Search Dialog Shortcuts

When the search dialog is open:

| Shortcut | Action |
|----------|--------|
| **Up arrow** | Move selection to the previous result |
| **Down arrow** | Move selection to the next result |
| **Enter** | Open the selected file |
| **Esc** | Close the search dialog |

## Browser Shortcuts

These are standard browser shortcuts that work naturally within Gleaner's SPA navigation:

| Shortcut | Action |
|----------|--------|
| **Alt+Left arrow** (Windows/Linux) / **Cmd+Left bracket** (Mac) | Navigate back |
| **Alt+Right arrow** (Windows/Linux) / **Cmd+Right bracket** (Mac) | Navigate forward |
| **Cmd+L** (Mac) / **Ctrl+L** (Windows/Linux) | Focus the browser address bar |
| **F5** / **Cmd+R** | Reload the page |

## Navigation Tips

Beyond keyboard shortcuts, there are several ways to navigate efficiently in Gleaner:

- **Click wikilinks** — Click any `[[wikilink]]` in the content area to jump to the linked file
- **Breadcrumbs** — Click any segment in the breadcrumb bar above the content to navigate up the directory tree
- **Browser back/forward** — Use your browser's back and forward buttons (or keyboard shortcuts) to move through your reading history
- **File tree** — Click any file in the left sidebar to open it directly
- **Graph nodes** — Click any node in the [Knowledge Graph](../features/Knowledge%20Graph.md) to navigate to that file
- **Backlink entries** — Click any entry in the right panel's backlinks list to jump to the linking file
