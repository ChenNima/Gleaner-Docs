# 多语言

Gleaner 界面支持中文和英文两种语言。

## 语言设置

在设置页面的「语言」选项卡中，你有三个选择：

| 选项 | 说明 |
|------|------|
| **跟随系统** | 根据浏览器/操作系统的语言设置自动选择 |
| **English** | 固定使用英文界面 |
| **中文** | 固定使用中文界面 |

默认为「跟随系统」，即自动检测系统语言。如果系统语言包含中文（`zh-CN`、`zh-TW` 等），界面显示中文；否则显示英文。

## 影响范围

语言设置影响所有 UI 元素：

- 导航栏按钮提示
- 设置页面文字
- 同步状态信息
- [[全局搜索|搜索框]] 占位符
- 错误提示信息
- [[知识图谱|图谱页面]] 标签

> 语言设置**不影响**文档内容的显示 — Markdown 内容按原样渲染。

## 持久化

语言偏好保存在浏览器的 `localStorage` 中，重启后保持不变。
