# 快捷键

Gleaner 支持键盘快捷键以实现高效导航。本页列出所有可用的快捷键和导航技巧。

## 全局快捷键

| 快捷键 | 操作 |
|--------|------|
| **Cmd+K** (Mac) / **Ctrl+K** (Windows/Linux) | 打开[全局搜索](../features/Search.zh.md)对话框 |

## 搜索对话框快捷键

当搜索对话框打开时：

| 快捷键 | 操作 |
|--------|------|
| **上方向键** | 将选择移动到上一个结果 |
| **下方向键** | 将选择移动到下一个结果 |
| **Enter** | 打开选中的文件 |
| **Esc** | 关闭搜索对话框 |

## 浏览器快捷键

以下是标准浏览器快捷键，在 Gleaner 的 SPA 导航中自然生效：

| 快捷键 | 操作 |
|--------|------|
| **Alt+左方向键** (Windows/Linux) / **Cmd+[** (Mac) | 后退 |
| **Alt+右方向键** (Windows/Linux) / **Cmd+]** (Mac) | 前进 |
| **Cmd+L** (Mac) / **Ctrl+L** (Windows/Linux) | 聚焦浏览器地址栏 |
| **F5** / **Cmd+R** | 刷新页面 |

## 导航技巧

除了键盘快捷键，还有多种方式在 Gleaner 中高效导航：

- **点击 wikilinks** — 在内容区中点击任意 `[[wikilink]]` 跳转到链接的文件
- **面包屑** — 点击内容上方面包屑导航栏中的任意段落，可向上导航目录树
- **浏览器前进/后退** — 使用浏览器的前进和后退按钮（或键盘快捷键）在阅读历史中移动
- **文件树** — 点击左侧边栏中的任意文件直接打开
- **图谱节点** — 在[知识图谱](../features/Knowledge%20Graph.zh.md)中点击任意节点导航到该文件
- **反向链接条目** — 点击右侧面板反向链接列表中的任意条目跳转到链接文件
