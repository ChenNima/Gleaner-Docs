# 快捷键

Gleaner 支持以下键盘快捷键，帮助你更高效地浏览内容。

## 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd+K` / `Ctrl+K` | [[全局搜索\|打开搜索]] | 在所有文件中搜索 |

## 搜索框内

在 [[全局搜索|搜索框]] 打开时：

| 快捷键 | 功能 |
|--------|------|
| `↑` | 选中上一条结果 |
| `↓` | 选中下一条结果 |
| `Enter` | 打开选中的文件 |
| `Esc` | 关闭搜索框 |

## 浏览器快捷键

以下是浏览器自带的快捷键，在 Gleaner 中同样有用：

| 快捷键 | 功能 |
|--------|------|
| `Alt+←` / `Cmd+[` | 返回上一个页面 |
| `Alt+→` / `Cmd+]` | 前进到下一个页面 |
| `Cmd+L` / `Ctrl+L` | 聚焦地址栏 |
| `F5` / `Cmd+R` | 刷新页面（重新同步） |

## 导航

在 [[文件浏览]] 页面中：

- 点击 [[双向链接|wikilink]] 跳转到目标文件
- 点击面包屑中的目录名展开文件树到对应位置
- 使用浏览器的前进/后退按钮在阅读历史中导航
