# 文档主题

Gleaner 提供多种 Markdown 渲染主题，改变文档的视觉呈现方式。

## 可用主题

| 主题 | 说明 |
|------|------|
| **GitHub** | 基于 GitHub Markdown 渲染的简洁样式。默认主题。 |
| **Obsidian** | 类似 Obsidian 笔记应用的样式。紧凑布局，独特的标题风格。 |
| **Academic** | 衬线字体与宽裕的行距，适合研究型长文阅读。 |
| **Notion** | 类似 Notion 的极简现代样式。 |
| **Newsprint** | 经典印刷风格的排版，温暖的纸张质感。 |

## 切换主题

1. 在文档内容上方的面包屑区域找到主题选择器
2. 从下拉菜单中选择主题
3. 主题会立即应用到当前文档

你的主题选择会自动保存，跨会话持久化。

## 深色 / 浅色模式

Gleaner 支持深色和浅色两种颜色模式，独立于 Markdown 主题：

- 使用顶部导航栏中的按钮在深色和浅色模式之间切换
- 模式会影响整个 UI（侧边栏、导航栏、背景），不仅限于内容区
- 你的偏好会自动保存

## 代码高亮

围栏代码块中的语法高亮会自动适配当前选择的主题和深色/浅色模式。无需单独配置——代码块始终使用在当前视觉环境下可读的颜色方案。
